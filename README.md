# COVID_Vaccine_Tracking
Uses CDC Vaccination Data to predict the number of Americans vaccinated on a given date.
The program pulls data from the CDC on vaccinations in the United States to determine the overall vaccination rate. The vaccination rate is used to model the number of Americans vaccinated on a specific date. The program runs by asking the user to input one of two variables: A) The percentage vaccinated. B) The date. The variable not selected by the user is ouput using data from the model.
